## About The Project

### Features

* Separating Red, Green, and Blue channels
* Modify brightness
* Modify contrast
* Turn image into grayscale
* Invert image color
* Scaling image
* Quantize image
* Filtering with kernel
* Image morphology operations
* Augment image using 2nd image
* Blending two images

### Built With

* Lazarus
